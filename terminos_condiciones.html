Términos y Políticas de Monkew Services

1. Definiciones Generales

"Vitrina": Sitio web personalizado dentro de Monkew Services para mostrar productos de un vendedor.

"Cliente": Persona o negocio que contrata los servicios de Monkew Services.

"Modificación": Cualquier cambio visual, textual o estructural solicitado.

2. Naturaleza del Servicio

Monkew Services ofrece vitrinas digitales alojadas en GitHub Pages. No incluye funciones de pagos ni carritos de compra.

3. Propiedad del Contenido

Todo contenido (imágenes, textos, precios) pertenece al cliente. El código y diseño pertenece a Monkew Services, salvo pago adicional para entrega de código.

4. Uso de GitHub Pages

Las vitrinas se alojan en GitHub Pages. Si el servicio se vuelve inviable, se migrará. Monkew no se responsabiliza por suspensiones por parte de GitHub.

5. Tarifas y Pagos

Los costos se dividen entre creación, mantenimiento y suscripción. La no renovación puede implicar desactivación sin aviso.

6. Reactivación y Cancelación

Una vitrina desactivada puede reactivarse pagando el mes de mantenimiento. No hay tarifa adicional por reactivación.

7. Descargo de Responsabilidad

Monkew Services no garantiza visitas, ventas ni interacción. La promoción es responsabilidad del cliente. Tampoco somos responsables por uso indebido de las vitrinas.

8. Entrega de Código

Los clientes pueden solicitar el código de su vitrina si pagan por su entrega. En ese caso, renunciamos a su mantenimiento.

9. Cambios y Actualizaciones

Las modificaciones fuera del plan base tienen costos establecidos. Las suscripciones incluyen un límite mensual.

10. Cambios a los Términos

Podremos actualizar estos términos. Los clientes activos serán notificados por correo.

Política de Privacidad

1. Información que Recopilamos

Recolectamos nombre, correo y redes sociales solo para contacto.

2. Uso de la Información

La información se utiliza solo para contacto y fines operativos.

3. Derechos del Cliente

El cliente puede solicitar la eliminación de su información en cualquier momento.
